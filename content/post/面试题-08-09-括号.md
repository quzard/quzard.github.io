## 难度（中等）

括号。设计一种算法，打印n对括号的所有合法的（例如，开闭一一对应）组合。

说明：解集不能包含重复的子集。

例如，给出 n = 3，生成结果为：

```
[
  "((()))",
  "(()())",
  "(())()",
  "()(())",
  "()()()"
]
```

<!--more-->

因为括号一定左括号先出
所以 只允许出现 `l == r` 与 `l < r `的情况

```c++
class Solution {
   vector<string> res;
public:
   vector<string> generateParenthesis(int n) {
      dfs(n, n, "");
      return res;
   }
   void dfs(int l, int r, string le){
      if(l ==0 && r == 0){
         res.push_back(le);
      }else{
         if(l != 0) dfs(l - 1, r, le + "(");
         if(l < r) dfs(l, r - 1, le + ")");
      }
   }
};
```
